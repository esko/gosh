# ADR 0002: Local Deltas Are Explicit

## Status

Accepted

## Decision

Only four product deltas are allowed without a new ADR:

- xterm.js `6.1.0-beta` with kitty keyboard protocol support.
- Arbitrary font support, including Nerd Fonts.
- Stronger theme, scrollback, and performance controls.
- Mosh support through upstream nassh/wassh.

## Consequences

Encrypted identities, fixture-specific UX, custom tab handling, bespoke dashboards, SFTP UI, jump hosts, and port forwarding are deferred unless approved separately.
